🏥 Gestión operativa de clínicas

Modelo demostrativo para revisar citaciones, tareas administrativas y carga operativa en recepción, sin utilizar datos sanitarios reales.

Uso local y demostrativo: Esta herramienta funciona en el navegador de quien la utiliza. Alonso & Espinosa no recibe, almacena ni accede a los datos introducidos. Usa siempre datos ficticios o de prueba. No introduzcas datos reales de pacientes, DNI, tarjeta sanitaria, teléfonos, correos reales, información médica ni datos especialmente protegidos. No sustituye a un software de gestión clínica ni a una historia clínica electrónica.

Para historia clínica electrónica, gestión clínica completa o tratamiento de datos sanitarios reales se requiere software específico y asesoramiento especializado.
